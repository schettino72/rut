# Examples

This directory contains example projects and use cases for rut.

## Basic Example

The `basic/` directory contains a minimal project showing:
- Simple test discovery
- Basic test structure
- Running tests with rut

## Async Example

The `async/` directory demonstrates:
- Writing async tests with `unittest.IsolatedAsyncioTestCase`
- Testing async functions
- Best practices for async testing

## Hooks Example

The `hooks/` directory shows:
- Using `conftest.py` for session-level setup/teardown
- Both sync and async hooks
- Database setup/teardown patterns

## Configuration Example

The `config/` directory demonstrates:
- Configuring rut via `pyproject.toml`
- Setting coverage sources
- Configuring warning filters
- Custom test base directory

## Coverage Example

The `coverage/` directory shows:
- Running tests with coverage
- Configuring coverage sources
- Interpreting coverage reports

Each example includes:
- A `README.md` with explanation
- Working test code
- Expected output
- Common patterns and best practices
